1. PROBLEM - I don’t see theme in System > Settings after uploading.

In OpenCart 2.2 and newest it was completly change to choose theme.

Go to Extension > themes > edit your store and choose theme from selector.

https://cl.ly/0v3y282T3N1F

2. PROBLEM - Log me out when open or edit Fastor Theme Options

  1. Please make sure you have newest theme version installed.
  2. Please make sure to upload “image” folder from our theme like described in “read first” document included in theme. Video how to do this here: https://cl.ly/312g3o1O0H29

3. PROBLEM - Revolution Slider doesn’t work with OpenCart 2.3

Unfortunately in OpenCart 2.3 there is ".htaccess" file in "system" folder which block revolution slider. You need to remove or change name.

We are also waiting for official Revolutsion Slider update and as soon as it will be available, we will update our theme also.

4. HOW TO - remove demo content?

Because our theme is multi-store, all demo content is added as new and not reaplaced. That allows you to manage other stores from one admin panel. You can remove old content on two ways:

  • You can reinstall modules in Extension > Modules: https://www.dropbox.com/s/97ksv7iceiyxl3k/Zrzut%20ekranu%202015-12-29%20o%2009.51.20.png?dl=0
  • Or remove new content manually in each module: https://www.dropbox.com/s/jk4zl7wytfsiudg/Zrzut%20ekranu%202015-12-29%20o%2009.51.31.png?dl=0

6. HOW TO - translate “years, months, days…” in countdowns

  1. Go to: catalog/view/theme/fastor/js/jquery.countdown
  2. Find: years, month, days…
  3. Translate in file

7. PROBLEM - First image on product page is smaller

You need to change imge size to bigger in OpenCart settings.

  • For OpenCart 2.0, 2.1 system/settings/images/product_thumb
  • For OpenCart 2.2 extension/themes/product_thumb

Settings in Fastor Theme Options allow you to increase/decrease photo area to make better responsive version of your store. You still need to change image size to bigger in OpenCart settings.
